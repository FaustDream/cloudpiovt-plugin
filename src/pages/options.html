推荐流程

1. 绑定当前页面目录

在云枢表单、列表或业务规则页面打开弹窗，点击"更新目标目录"，为当前页面类型建立独立目录快照。

2. 前端先抓取再回写

表单页写入 form-index.html / form-style.css / form-script.js；列表页写入 list-index.html / list-style.css / list-script.js。本地修改后用"前端回写"同步。

3. 业务规则保持单页单 model

进入业务规则页后先抓取 .java 文件，本地改完再回写；同一页面不要同时打开多个业务规则，避免命中错误 Monaco model。

4. 用协作文件对齐上下文

抓取会按需补齐 README.md、AGENTS.md、DESIGN.md，并刷新 FromCode.md；云枢 JS 通过业务规则传参协作。

内置规则说明

    推荐流程

    1. 打开氚云表单设计器

    进入 h3yun.com 表单设计页面后切到弹窗"氚云"标签，先确认页面已经加载图形控件、前端 JS 或后端 C# 区域。

    2. 使用一键抓取写入

    一次补齐 README.md、AGENTS.md、DESIGN.md、FromCode.md，并写入表单 ID 对应的 .js 文件和 C# 类名对应的 .cs 文件；未挂载区域会跳过。

    3. 在本地维护设计与代码

    README.md 用于用户填写需求，DESIGN.md 用于 AI 整理实现逻辑和代码文件，FromCode.md 用于核对字段编码；氚云 JS 和 C# 通过 Ajax 互通。

    4. 分别回写前端和后端

    前端代码回写读取同表单 ID 的 .js 文件，后端代码回写读取 C# 类名 .cs 文件，按当前页面 Monaco model 写回。
