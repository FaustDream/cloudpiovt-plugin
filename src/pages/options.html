推荐流程

1. 在云枢表单/列表页打开弹窗，点击"更新目标目录"建立页面目录绑定。

2. 前端抓取写入：表单页 → form-* 文件；列表页 → list-* 文件。本地修改后用"前端回写"同步。

3. 业务规则：进入后先抓取 .java，改完再回写；同一页面不要同时打开多个业务规则。

4. 协作文件：抓取按需补 README/AGENTS/DESIGN/FromCode；云枢 JS 通过业务规则传参协作。

内置规则

    控件参考

    控件类型中文名称说明

    推荐流程

    1. 打开氚云表单设计器页面，确认前端 JS 或后端 C# 区域已加载。

    2. 一键抓取写入：读取控件信息、前端 JS、后端 C#，并补齐协作文件；未挂载区域自动跳过。

    3. 本地维护：README 写需求，DESIGN 整理实现逻辑，FromCode.md 核对字段编码；JS/C# 通过 Ajax 互通。

    4. 分别回写：前端 .js 通过 Monaco model API 写回 #jsText；后端 .cs 写回 #csText。
