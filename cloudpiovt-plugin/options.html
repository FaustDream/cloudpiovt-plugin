    使用文档

    首次配置和日常使用建议按下面顺序执行，能避免目录未绑定、编辑器打不开或回写命中错误页面。

    1. 初始化

    先配置 VS Code 与 IDEA 的 exe 路径，再回到弹窗点击“更新目标目录”，为当前页面类型绑定本地目标目录。

    2. 前端页面

    使用“前端抓取写入”把页面中的 html、css、javascript 写入目标目录；本地修改后，再用“前端回写”同步回页面。

    3. 业务规则

    使用“业务规则抓取写入”导出当前 Monaco 中的 .java 文件；本地修改后，用“业务规则回写”按同名文件回写到对应 Monaco model。

    4. 快捷操作

    弹窗顶部的 VS Code / IDEA 按钮会直接打开当前页面绑定的目标目录；“复制链接”和“复制路径”可复制完整值用于排查和分享。
