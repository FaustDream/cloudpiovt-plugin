# 2026-06-10

## Bug修复：氚云后端代码回写失败 (preflightResults is not defined)

- **问题**：氚云平台「后端代码回写」和「前端代码回写」均失败，日志报 `ReferenceError: preflightResults is not defined`
- **根因**：`popup.js:2053` 行，`handleH3yunCodeWriteback` 函数中 `runOperationWithPreflight` 的回调参数解构缺少 `preflightResults`，而第 2071 行调用了它
- **修复**：将解构 `{ pageContext, directorySelection }` 改为 `{ pageContext, directorySelection, preflightResults }`，与云枢业务规则回写（第 1903 行）保持一致

## Bug修复：设置页/扩展页打开资源管理器被 PAGE_NOT_SCRIPTABLE 阻断

- **问题**：在扩展设置页（chrome-extension://）点击打开资源管理器，预检报 `PAGE_NOT_SCRIPTABLE` 阻断
- **根因**：`runPageContextPreflight` 无条件检查 `getExecutableContextError`，而打开外部程序不需要向页面注入脚本
- **修复**：给 `runPageContextPreflight` 增加 `skipExecutableContextCheck` 参数；`runOperationWithPreflight` 透传 `options.skipExecutableContextCheck`；`handleOpenCustomLauncher` 传入 `skipExecutableContextCheck: true`
