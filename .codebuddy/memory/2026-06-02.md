# 2026-06-02 工作日志

## 修复 cloudpiovt-plugin  Native Host 安装脚本编码问题

- **问题**：`install-native-host.cmd` 运行报错，`cmd.exe` 将中文标点 `，` `。` 的 UTF-8 字节序列解析为多个命令碎片。
- **根因**：`.cmd` 文件含非 ASCII 字符（中文标点），`cmd.exe` 对 UTF-8 without BOM 的解析存在兼容性问题。
- **修复**：移除所有非 ASCII 字符，改为英文提示。文件现为纯 ASCII（UTF-8 without BOM）。
- **文件**：`cloudpiovt-plugin/scripts/install-native-host.cmd`
- **建议**：如需中文提示，需存为 UTF-8 with BOM 并确保 `chcp 65001` 生效，但兼容性较差，建议保持英文提示。
